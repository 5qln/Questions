# Security Policy

This repository is a public, CC0 commons of *questions*. Its one security
property is the **membrane**: no private content from an /idk cycle (α, Z, ∇, B″,
the Codex seal, cycle numbers, identities) may ever appear here. A leak of any
such content — or a way to bypass `bin/membrane_lint.py` — is a security issue.

## Reporting

Email **security@5qln.com** with the file, the commit, and what leaked or how the
gate was bypassed. Please do **not** open a public issue for a suspected leak;
that would republish the very content we are trying to contain.

- Acknowledgement within **3 business days**.
- If a leak is confirmed, the offending content is removed and its hash rotated.
